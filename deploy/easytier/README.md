## Setup EasyTier on Linux

```shell
export EASYTIER_COMMAND="-d --network-name finalrip --network-secret finalrip -e tcp://easytier.public.kkrainbow.top:11010"
curl -L https://raw.githubusercontent.com/TensoRaws/FinalRip-dev/main/deploy/easytier/easytier.sh -o easytier.sh && chmod +x ./easytier.sh && ./easytier.sh
```
